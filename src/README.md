## Documentation

To view the [Doxygen](https://doxygen.nl/) generated documentation:

### Windows
1. Navigate to the `doxygenDocs/html` folder
2. Double-click `index.html` to open in your default web browser

### Mac
1. Navigate to the `doxygenDocs/html` folder
2. Double-click `index.html` to open in your default web browser
   ```bash
   open doxygenDocs/html/index.html
   ```
